# Introductory-SQL-Project-Maven-Music-Database
This project demonstrates end-to-end SQL skills by designing, managing, and analyzing a relational database for a fictional music streaming platform called Maven Music. The script covers database creation, data cleaning, table relationships, and insightful analytical queries.

# Key Components

Database Design: Created normalized tables for customers, audio files, login sessions, and listening history.

Data Cleaning & Transformation: Applied SQL functions to clean email data, fix null values, and standardize date formats.

Relational Modeling: Established foreign key relationships for customer sessions and audio data integrity.

Exploratory Analysis:

Top genres and most-played songs

Active vs. cancelled subscribers

Listening trends by customer and genre

Premium user preferences and churn analysis

# SQL Concepts Used

CREATE, ALTER, UPDATE, and JOIN statements

Aggregate functions (COUNT, AVG, ROUND)

Subqueries and GROUP BY clauses

Date operations (DATEDIFF, STR_TO_DATE)

Data validation and missing value checks

# Objective

To demonstrate SQL proficiency in database creation, cleaning, and analytics by simulating a real-world dataset of a music streaming platform.
